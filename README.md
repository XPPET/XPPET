### Πώς θα το δεις να δουλεύει ΤΩΡΑ:
Το GitHub θα το τρέξει μόνο του το βράδυ, αλλά μπορείς να το αναγκάσεις να τρέξει τώρα:
1. Πήγαινε στην καρτέλα **Actions** στο πάνω μέρος του repository.
2. Αριστερά πάτα το **generate animation**.
3. Πάτα το κουμπί **Run workflow** -> **Run workflow**.
4. Περίμενε 1 λεπτό να τελειώσει και μετά ανανέωσε το README σου!

Θέλεις να προσθέσουμε και μια λίστα με τα **Latest Projects** σου που θα ενημερώνεται κι αυτή αυτόματα;
